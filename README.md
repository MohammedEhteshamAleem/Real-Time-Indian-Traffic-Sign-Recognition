# Real-Time-Indian-Traffic-Sign-Recognition
Deep learning based real-time traffic sign recognition system using YOLO designed for complex Indian traffic environments.

## Project Overview

Traffic sign detection and recognition are crucial components of intelligent transportation systems and autonomous vehicles. This project presents a real-time traffic sign recognition system designed specifically for complex Indian urban traffic environments using the YOLOv9 deep learning algorithm.  
The system detects and classifies traffic signs from live video streams with high accuracy and real-time performance, enabling safer and smarter traffic management.

## Problem Statement

Indian road environments are highly dynamic and complex. Traditional traffic sign recognition systems struggle due to:  
• Lighting variations  
• Occlusion of traffic signs  
• Complex urban backgrounds  
• Region-specific traffic signs  

This project aims to develop a robust and scalable real-time traffic sign recognition system capable of handling these challenges.

## Proposed System

This project introduces a YOLOv9-based real-time traffic sign recognition framework optimized for Indian road conditions.
Key components include:  

1. Dataset Creation and Annotation:  
• Collection of Indian traffic sign images  
• Manual annotation for training  
2. Model Training:  
• Fine-tuning YOLOv9 using transfer learning  
• Anchor box optimization  
3. Data Augmentation:  
• Rotation  
• Scaling  
• Brightness adjustment  
• Occlusion simulation  
4. Integration with Smart Traffic Systems:  
• Real-time detection from live video feeds  
• Potential integration with ADAS and traffic management systems
